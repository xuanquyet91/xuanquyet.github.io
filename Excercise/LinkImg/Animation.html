Chú Ý:: Thuộc tính opacity mang tính kế thừa, tức là nếu element cha có opacity là 0.8 thì element con sẽ có opacity mặc định là 0.8. Khi đó nếu lại set opacity của element con là 0.5 vậy opacity thực tế sẽ là 0.8 x 0.5 = 0.4
