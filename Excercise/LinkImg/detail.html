CSS và cách sử dụng trên HTML

CSS: Cascading style sheets.
CSS dùng để trình bày nội dung, có 3 cách trình bày:

Inline style: Nhúng trực tiếp vào trong các Tag.
Internal Style: Sử dụng thẻ style nhúng trực tiếp trong tag Head.
Enternal Style: Sử dụng thẻ link liên kết bên trong vs bên ngoài bằng CSS

**Lưu Ý: nên dùng Enternal style.

Excercise:
Đâu là cú pháp đúng để nhúng file style.css vào trang HTML?: link rel="stylesheet" type="text/css" href="style.css"

Để sử dụng Internal style & inline ta phải dùng thẻ nào?: style

Để sử dụng External style ta phải dùng thẻ nào?: LInk

CSS Selectors

Có rất nhiều selectors: có 3 loại chính: Element, ID, Class.
Group Selector năng cách bởi dấu phẩy( Nhóm các selector cùng style).
Cú pháp: #iD và .Class (ID chỉ dùng một lân, còn class dùng không giới hạn)
Đặt tên chi ID & Class không bắt đầu bằng số, và dùng tiếng việt có dấu.
Mức độ ưu tiên của selectors( khi style trùng nhau giảm dần):
Inline style -> ID selector => class selector => element selector. Một số Selector hay dùng:
* : Chọn tất cả element.
.box1, .box2 : Chọn toàn bộ element có class="box1" và toàn bộ element có class="box2".
.box1.box2 : Chọn những element mà có đồng thời cả 2 class "box1" và "box2" (class="box1 box2")
.box1 .box2 : Chọn toàn bộ element có class="box2" mà nằm bên trong element có class="box1"
.box1 > .box2 : Chọn những element có class="box2" mà bao quanh trực tiếp bởi element có class="box1".

Styling Text

Các Attribute thường dùng:
- Text-align: căn vị trí text- trái-phải-giữa- đều hai bên.
- Text-transformation : Chuyển chữ sang dạng viêt hoa, thường, hoặc chữ cái đầu viết hoa.
- Text-decoration: Thêm dấu gạch ngang, gạch chân...
Font-size: Cỡ chữ. Tiêu đề thông thường có cỡ chữ gấp 1,5 lần font-size.

Font-style: Xác định loại chữ..
font-variant: Chuyển đổi kiểu chữ (thường thành hoa).
font-weight: Hiển thị chữ đậm hay thường.
font-family: Xác định giá trị font family cho chữ: arial...

Đơn vị đo lường trong CSS:
- Px: Pixels( Điểm ảnh trên màn hình máy tính).
- Em(tương đương với kích thước của font-size hiện dùng). VD: Font-size 16px => 1 em= 16px.
- %: Áp dụng được cho nhiều giao diện.
